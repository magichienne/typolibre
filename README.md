# typolibres

Liste de ressources typographiques libres : fonderies, typothèques, fontes et outils gratuits et open source.

## fonctionnalités

- tri par nom, type et licence
- filtrage par tags
- mode accessible (contraste élevé)
- responsive (mobile < 800px)
- tri insensible aux accents

## technologies

- PHP
- JavaScript (vanilla)
- CSS3 (variables CSS, CSS Grid/Flexbox)

## installation

1. cloner le dépôt
2. servir avec un serveur PHP :

```bash
php -S localhost:8000
```

3. ouvrir http://localhost:8000

## polices utilisées

- **Alte Haas Grotesk** — [Yann Le Coroller](https://www.dafont.com/alte-haas-grotesk.font), SIL Open Font License
- **Redaction** — [Forest Young & Jeremy Mickel](https://www.redaction.us/), SIL Open Font License
